What is SearchSift?

SearchSift is a local-first tool that captures your search queries and search result clicks, automatically categorizes them, and produces insightful daily reports. All data stays on your local machine.

Getting Started

  1. Install the browser extension
  2. Configure your API key in extension settings
  3. Start searching - your queries are automatically captured
  4. View the dashboard for real-time statistics
  5. Generate daily reports for deeper analysis

Categories

SearchSift automatically categorizes your searches using keyword matching:

Work Coding Research Shopping Social News Entertainment Finance Health Travel Other

Privacy

SearchSift captures only:

  • Search query text
  • URLs of clicked search results
  • Timestamps
  • Search engine name

We do NOT capture: page content, keystrokes, browsing history outside search engines, or any personal information beyond your search queries.

Troubleshooting

Extension not connecting?

  1. Verify the backend is running: curl http://127.0.0.1:5000/health
  2. Check API key matches in extension options and config.py
  3. Look for errors in the extension console (right-click icon → Inspect)

No searches being captured?

  1. Ensure content script is loaded (check extension details)
  2. Verify search engine URL matches patterns in manifest
  3. Check logs at logs/searchsift.log

API Endpoints

Endpoint Description
